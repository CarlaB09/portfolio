# Carla Vasconcelos - Portfolio Website
A modern, responsive portfolio website showcasing my skills and projects as a software developer. Built with HTML, CSS, and JavaScript, featuring a clean design with interactive elements.

[Live page link](https://carlab09.github.io/portfolio/)

#### 🌟 Features
* Responsive Design: Fully responsive layout that adapts to all device sizes

* Video Background: Dynamic background video with text overlay

* Interactive Navigation: Smooth-scrolling navbar with active page indicators

* Contact Form: Integrated contact form using Formspree for message handling

* Image Carousel: Automatic slideshow showcasing technical skills

* Social Links: Direct links to GitHub and LinkedIn profiles

* Smooth Animations: CSS animations and transitions throughout

<br>

#### 🛠️ Technologies Used

* HTML5

* CSS3

* JavaScript (ES6+)

* Font Awesome

*Formspree (form handling)
